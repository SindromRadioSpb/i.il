# docs/EDITORIAL_STYLE.md — RU Editorial Style Guide

This guide standardizes tone, structure, and formatting for Russian posts and website stories.

## 1) Tone
- Neutral, factual, non-sensational.
- No moral judgments or speculation.
- Prefer “по данным источников” when facts are still developing.

**Forbidden (examples):**
- “ужас”, “кошмар”, “шок”, “потрясающе”, “скандал века” (unless it is a direct quote that is essential)
- clickbait punctuation like “!!!” or “Срочно!!!”

## 2) Mandatory structure (for every Story)
**Title (1 line):** factual and short  
**Что произошло:** 1–2 sentences  
**Почему важно:** 1 sentence  
**Что дальше:** 1 sentence or “Ожидается обновление.”  
**Источники:** 2–5 links (when available)

## 3) Length targets
- Summary body (without “Источники”): **400–700 characters**.
- Facebook post body: 1 title + 2–4 bullets, short.

## 4) Numbers, dates, names
- Preserve numbers exactly (casualties, sums, percentages).
- Date/time in RU format, keep internal original for verification.
- Names/toponyms: follow `docs/GLOSSARY.md` rules.

## 5) Risk-level specific rules
### `risk_level=high` (casualties, terror, emergencies, health)
- Strict neutrality.
- Add “по данным источников” early in the summary.
- Prefer ≥2 sources when available.
- No graphic details.

### `risk_level=medium` (politics/courts)
- Avoid strong framing.
- Clearly separate facts from commentary.

## 6) Attribution
Always list sources with links.
If only one source exists:
- include it
- add “данные уточняются” when appropriate

## 7) Updates
If story updates materially:
- update summary text
- optionally add “Обновлено: HH:MM” line on the hub page (not required for Facebook)
